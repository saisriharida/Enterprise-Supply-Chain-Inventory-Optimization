 🏪 Enterprise Supply Chain & Inventory Optimization Platform

 📌 Project Overview
This project analyzes customer purchasing behavior and inventory demand patterns using the **Instacart Market Basket Analysis** dataset. The goal is to help business stakeholders optimize inventory planning, improve product availability, and identify high-demand products through data-driven decision making.

The project follows a complete Data Analyst workflow:
- Data Collection & Validation
- SQL-based Business Analysis
- Python Data Cleaning & Exploratory Data Analysis (EDA)
- Interactive Power BI Dashboarding
- Business Insights & Recommendations

---

 🎯 Business Problem

Large retail and e-commerce companies need to answer questions like:
- Which products should always remain in stock?
- Which departments generate the highest demand?
- What products are frequently reordered by customers?
- During which days and hours does order volume peak?
- How can inventory and workforce planning be optimized?

This project addresses these business challenges using transactional order data.

---

 📂 Dataset
**Dataset:** Instacart Market Basket Analysis

 Tables Used
| Table | Description |
|--------|-------------|
| orders | Customer order history and timing information |
| order_products__prior | Products purchased in previous orders |
| products | Product master data |
| aisles | Product aisle categories |
| departments | Product department categories |
